## Program Details

### Core Information
- **Program Name**: AI Robotics Curriculum: From Zero to Hero
- **Duration**: 40 weeks (10 months)
- **Schedule**: Fridays, 2+ hours per session
- **Target Audience**: Grades 8-11 (Ages 13-17)
- **Prerequisites**: None - designed for complete beginners
- **Class Size**: 10-15 students optimal
- **Monthly Fee**: R1500/month

### Educational Philosophy
- **Zero to Hero Approach**: Start from absolute beginner knowledge
- **Hands-On Learning**: Learn by doing, not memorizing
- **AI-Assisted Development**: Use AI tools as learning assistants
- **No Grades System**: Focus on confidence building and project completion
- **Tangible Outcomes**: Every week results in a working project

## Pricing and Equipment Structure

### Monthly Investment: R1500/month

#### Phase 1 Equipment Included (Weeks 1-8)
- **Digital Foundations**: Raspberry Pi 4 starter kit
- **Basic Electronics**: LED kits, resistors, buttons, sensors
- **Tools**: Digital multimeter, screwdriver set
- **Learning Materials**: All software and educational resources

#### Phase 2 Equipment Included (Weeks 9-16)
- **Robotics Components**: Robot chassis kit with wheels
- **Movement Systems**: Motors, servos, motor controllers
- **Sensor Kit**: Light, temperature, distance sensors
- **Building Supplies**: Construction materials for first robot

#### Phase 3 Equipment Included (Weeks 17-28)
- **AI Vision**: Raspberry Pi camera module
- **Audio Components**: Microphone module, speakers
- **Advanced Sensors**: Motion, accelerometer, color sensors
- **Display Systems**: OLED/LCD screens for projects

#### Phase 4 Equipment Included (Weeks 29-40)
- **Specialized Components**: Custom project-specific parts
- **Construction Materials**: Advanced building supplies
- **Project Enclosures**: Custom housing for final prototypes
- **Additional Electronics**: Based on student project choices

### Additional Cost Transparency
- **Potential Extra Costs**: R200-500 for specialized final project components
- **Supplier Information**: Detailed supplier lists and pricing provided
- **Optional Upgrades**: 3D printed parts, premium components available
- **Equipment Ownership**: Students keep all components as they progress

# Project Templates and Worksheets
*Ready-to-use templates for every week of the AI Robotics Curriculum*

## Overview

This document provides printable templates, worksheets, and project guides for each week of the 40-week curriculum. These templates help students stay organized, document their learning, and create consistent project portfolios.

## Template Philosophy

- **Consistent Structure**: Similar format each week for familiarity
- **Progressive Complexity**: Templates grow with student skills
- **Documentation Focus**: Emphasize recording the learning process
- **Reflection Integration**: Built-in opportunities for student reflection
- **Portfolio Building**: Each completed template becomes part of student portfolio

## How to Use These Templates

### **For Students**
1. **Complete Weekly**: Fill out each template during or after class
2. **Be Honest**: ShareÁúüÂÆûÁöÑ challenges and learning
3. **Add Details**: Include specific code snippets, measurements, and observations
4. **Take Photos**: Document your progress with images
5. **Keep Organized**: Store completed templates in your portfolio folder

### **For Instructors**
1. **Print Copies**: Have templates ready for each class
2. **Review Weekly**: Use templates to track student progress
3. **Provide Feedback**: Write comments and suggestions
4. **Adapt as Needed**: Modify templates for your specific context
5. **Create Portfolio**: Compile completed templates into final portfolios

### **For Portfolio Building**
1. **Keep All Templates**: Every completed template shows growth
2. **Add Photos**: Include pictures of projects and work
3. **Write Reflections**: Add personal insights to each template
4. **Organize Chronologically**: Show progression over 40 weeks
5. **Share Achievements**: Use portfolios for demonstrations and applications

These templates provide structure for documenting the learning journey while allowing flexibility for creativity and individual expression. They become a comprehensive record of student growth from complete beginners to confident creators.

# Weekly Session Plans: AI Robotics Curriculum
*Detailed 2-hour Friday session breakdowns for instructors and students*

## Session Structure Template
Each 2-hour Friday session follows this consistent structure:
- **0:00-0:30 (30 min)**: Introduction, review, and demonstration
- **0:30-1:30 (60 min)**: Hands-on building and coding
- **1:30-2:00 (30 min)**: Testing, demonstration, and reflection

---

## Phase 1: Digital Foundations (Weeks 1-8)

### Week 1: Computer Confidence
**Theme**: Computers aren't scary - they're tools that help us create!

**0:00-0:30: Introduction**
- Welcome and icebreaker: "What's the coolest thing you've seen a computer do?"
- Computer anatomy: What's inside the box? (CPU, RAM, storage in simple terms)
- Demonstration: File organization - creating a clean digital workspace
- Today's goal: Create your own digital portfolio system

**0:30-1:30: Hands-On Activity**
- Create folder structure: "AI_Robotics_2024" with weekly folders
- Practice saving, finding, and renaming files
- Learn keyboard shortcuts (Ctrl+C, Ctrl+V, Ctrl+Z, Alt+Tab)
- Take a "tech inventory" photo and save it properly

**1:30-2:00: Demonstration & Reflection**
- Students show their organized folder systems
- Discuss: "Why does being organized help us be creative?"
- Preview: Next week we'll make our first interactive stories!
- Homework challenge: Find and organize 5 random files on your computer

**Materials Needed**: Computers, mouse, keyboard, internet access

## Teaching Notes and Tips

### **For Instructors**
- **Flexibility is Key**: Adapt timelines based on student progress and interests
- **Celebrate Small Wins**: Every completed project is a major achievement
- **Encourage Peer Teaching**: Students learn deeply by helping others
- **Document Everything**: Photos and videos create powerful learning records

### **AI Integration Guidelines**
- **AI as Assistant**: Use AI to enhance, not replace, student thinking
- **Prompt Engineering**: Teach students how to ask effective questions
- **Ethical Considerations**: Discuss responsible AI use throughout
- **Critical Thinking**: Always ask "Is this AI response reasonable?"

### **Differentiation Strategies**
- **Multiple Project Options**: Various ways to demonstrate learning
- **Extension Activities**: Challenges for advanced students
- **Peer Support**: Create opportunities for collaborative learning
- **Individualized Pacing**: Allow students to progress at appropriate speeds

### **Assessment Philosophy**
- **No Grades**: Focus on growth, effort, and completion
- **Portfolio-Based**: Document journey and achievements
- **Self-Reflection**: Students assess their own learning and growth
- **Celebration**: Every completed project deserves recognition

This curriculum transforms complete beginners into confident creators through carefully designed, hands-on projects that build technical skills while developing creativity, problem-solving abilities, and the confidence to tackle new challenges.

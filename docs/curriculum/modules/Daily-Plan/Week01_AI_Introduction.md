### Activity 1: AI-Generated Personal Website (20 min)
**The Challenge:** "Create a website that tells the world about YOU"

**Step-by-Step Guide:**
1. **Prompt Engineering Practice:**
   ```
   "Help me create a personal website. I am [age] years old and I love [hobbies].
   Make it cool and colorful with HTML and CSS."
   ```

2. **Copy, Paste, Customize:**
   - Students get AI-generated code
   - Save as `index.html` in VS Code
   - Change colors, text, add personal details
   - Ask AI for help: "How do I change the background color?"

3. **Make It Work:**
   - Open the HTML file in browser
   - "WOW! It's actually working!"
   - High-fives and celebration

**Teacher Tip:** Walk around and help students craft better AI prompts. Celebrate every working website!

# Module 2: Building with AI Agents (8 Weeks)
*One 3-hour session per week*

## Week 1: Introduction to AI Agents
### Session Structure (3 hours)
- **Introduction (30 min)**
  - Types of AI agents
  - Agent capabilities
  - Real-world applications

- **Hands-on Workshop (2 hours)**
  - Testing different AI agents
  - Basic agent configuration
  - Simple task automation
  - Agent comparison exercise

- **Project Launch (30 min)**
  - Weekly project: Create a personal assistant agent
  - Planning and requirements

### Take-Home Activities
- Set up development environment
- Begin agent development
- Document agent behavior

## Week 2: Personal Assistant Agents

- **Project Development (30 min)**

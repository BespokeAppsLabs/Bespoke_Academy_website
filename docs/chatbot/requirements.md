# Requirements

## Overview
Bespoke Academy is designed to be accessible to all students regardless of their technical background. We provide all necessary equipment and software, with minimal requirements for participation.

## Key Information
- No prior experience required
- Basic computer sufficient for early phases
- All specialized equipment provided
- Software licenses included
- Technical support available
- Setup assistance provided

## Common Questions
**Q: Does my child need a powerful computer?**
A: No! A basic computer is sufficient for the first phases. We provide specialized equipment when needed.

**Q: What software do we need to purchase?**
A: None! All software licenses are included in the tuition and provided by the academy.

**Q: What if we don't have technical experience at home?**
A: No problem! We provide complete technical support and setup assistance for all families.

## Detailed Information

### Computer Requirements
- **Basic Computer**: Any modern computer (Windows, Mac, or Chromebook)
- **Internet Access**: Standard home internet connection
- **Web Browser**: Current version of Chrome, Firefox, or Safari
- **Webcam/Microphone**: For virtual components and communication

### Software Provided
- All programming environments and tools
- Design and simulation software
- Project management platforms
- Communication and collaboration tools
- Cloud-based development environments

### Equipment Provided (as needed)
- Robotics kits and components
- Electronic components and sensors
- Specialized tools and equipment
- Project materials and supplies
- Safety equipment and gear

### Technical Support
- Initial setup and installation assistance
- Ongoing troubleshooting support
- Equipment loan program if needed
- Remote assistance options
- Parent technical training sessions

### Progressive Requirements
- **Phase 1**: Basic computer only
- **Phase 2**: We provide robotics kits
- **Phase 3**: Access to cloud-based AI tools
- **Phase 4**: All capstone project materials included

## Agent-Specific Assessment Tasks

### 1. Bespoke Education Minister Agent Assessment

**Focus**: Curriculum quality and educational effectiveness

**Key Areas to Evaluate**:
- Review `/docs/curriculum/framework.md` and 40-week curriculum structure
- Assess age-appropriateness for Grades 8-11 (ages 13-17)
- Evaluate learning progression and scaffolding across 4 phases
- Analyze assessment methods and outcome alignment

4. How well does the AI-assisted learning methodology align with educational best practices?
5. What gaps exist in the current curriculum framework?

**Evidence Sources**:
- `/docs/curriculum/framework.md`
- `/docs/curriculum/modules/` directory
- `/docs/curriculum/assessment_feedback_system.md`
- `/docs/curriculum/ai_personalization_framework.md`

## Coordination Instructions

### For Each Agent:
1. **Read the Assessment Framework**: `/v3/assessment-framework.md`
2. **Review Your Domain**: Examine specified evidence sources
3. **Follow Template**: Use the standardized assessment format
4. **Provide Evidence**: Reference specific files and examples
5. **Score Readiness**: Provide 1-10 domain readiness score
6. **Recommend Actions**: Prioritize immediate, short-term, and long-term actions

### Submission Process:
1. Complete your assessment following the template
2. Submit findings as a response to this coordination brief
3. Include specific file references and evidence
4. Provide actionable recommendations with timelines
5. Identify critical risks that need immediate attention

### Compilation:
All assessments will be synthesized into:
- `/v3/state.md` - Comprehensive project state report
- Consolidated action plan with priorities
- Risk matrix with mitigation strategies
- Success metrics dashboard
- Stakeholder presentation summary

## Success Criteria

**Assessment Quality**:
- Comprehensive domain coverage

- Actionable recommendations

- Realistic timelines and priorities

**Integration Excellence**:
- Seamless compilation of all domains
- Identification of cross-domain dependencies
- Unified success metrics and KPIs
- Cohesive action plan alignment
- Stakeholder-ready final report

**Timeline**:
- Individual assessments: Complete within current session
- Compilation and synthesis: Immediate following session
- Final state report: Ready for stakeholder review
